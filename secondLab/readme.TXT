Класс atribute_skill (класс навыка таких как скрытность, религия, атлетика и т.д): 
    _name          : имя навыка
    _val           : значение навыка
    _atr_val       : значение атрибута, к которому относится навык
    _mastered      : является ли навык освоеным (прибалять ли бонус мастерства)
    _mastered_val  : значение бонуса мастерства
	
	set_mastered_val()	: установить значение бонуса мастерства
	get_val()			: получить значение навыка
	get_name()			: получить имя навыка
	get_mastered()		: получить освоен ли навык
	
Класс atribute (класс атрибут такой как сила, ловкость и т.д): 
	_name           	: имя класса
    _val            	: значение атрибута
    _bonuse         	: бонус
    _mastered       	: освоенли навык
    _mastered_val   	: бонус мастерства
    _modificator    	: модификатор атрибута
    _val_to_modificator : словарь для значения модификатора
    _atribute_skills 	: словарь навыков
	
	set_modificator()		:	установить модификатор
	get_modificator()		:	получить модификатор
	get_rand_save_value()	:	получить случайное значение спасброска
	set_atribute_skill()	:	добавить навык
	get_skill()				:	получить навык
