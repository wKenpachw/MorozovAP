16) структра проекта на Python. Модули. Иерархия классов.
Объектно-ориентированная парадигма программирования (и, в частности, принцип наследования)
подразумевает построение иерархии классов, которая бы в совокупности наиболее эффективным образом решала поставленную задачу.
В Python все классы наследуются от класса object.
Встроенные классы Python также имеют свою иерархию, например, для типа bool она выглядит как:

object -> int -> bool
